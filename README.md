## 🏛️ Official & Core (官方核心标准)
*最权威的基础设施与标准实现。*

- [**anthropics/skills**](https://github.com/anthropics/skills) - Anthropic 官方提供的标准 `SKILL.md` 范例库，包含文件读取、代码分析等基础能力。
- [**modelcontextprotocol/servers**](https://github.com/modelcontextprotocol/servers) - MCP 协议官方参考实现，提供 Git, Filesystem, Slack 等核心连接能力。
- [**github/awesome-copilot**](https://github.com/github/awesome-copilot) - GitHub 官方维护列表，关注其中的 `/skills` 子目录。

## 📚 Collections (综合集合与列表)
*其他开发者维护的优质资源列表。*

- [**appcypher/awesome-mcp-servers**](https://github.com/appcypher/awesome-mcp-servers) - 社区维护最全的 MCP 服务器列表，涵盖 DevOps、金融、安全等多个领域。
- [**wong2/awesome-mcp-servers**](https://github.com/wong2/awesome-mcp-servers) - 由知名开发者整理的 MCP 工具清单，偏向实用性。

## 💻 Development & DevOps (开发与运维)
*赋予 Agent 编写代码、管理服务器的能力。*

- [**CoplayDev/unity-mcp**](https://github.com/CoplayDev/unity-mcp) - **Game Dev**: 允许 Agent 直接控制 Unity 编辑器进行游戏开发。
- [**flux159/mcp-server-kubernetes**](https://github.com/flux159/mcp-server-kubernetes) - **Infra**: Kubernetes 集群管理技能，支持 Pod 监控与部署。

## ➕ How to Contribute (如何提交)

我们非常欢迎社区贡献！如果你发现了好用的 Skill，请通过以下方式提交：

1. **推荐方式**：点击 [这里提交 Issue](https://github.com/chuangmeet/AgentSkillHub/issues/new?template=submit_skill.yml) (需要先创建 Issue 模板)。
2. 或者直接提交 Pull Request 修改此文件。
