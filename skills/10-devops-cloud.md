# ☁️ DevOps & Cloud (云原生与运维)

[🔙 返回首页](../README.md)

> 面向工程师、SRE 和架构师的硬核技术工具，涵盖云成本、容器编排与基础设施代码。

## 资源列表

### Cloud & FinOps (云成本管理)
- [**aws-cost-optimization**](https://github.com/ahmedasmar/devops-claude-skills/tree/main/skills/aws-cost-optimization)
    - **简介**: AWS 成本优化综合工作流。支持查找闲置资源、分析预留实例 (RI)、检测异常及生成月度报告。
    - **标签**: `AWS`, `FinOps`, `Cost Management`

### Kubernetes & GitOps
- [**k8s-troubleshooter**](https://github.com/ahmedasmar/devops-claude-skills/tree/main/skills/k8s-troubleshooter)
    - **简介**: 系统化的 Kubernetes 故障排查工具。诊断 Pod 启动失败、CrashLoopBackOff 及集群健康问题。
    - **标签**: `Kubernetes`, `SRE`, `Debugging`
- [**gitops-workflows**](https://github.com/ahmedasmar/devops-claude-skills/tree/main/skills/gitops-workflows)
    - **简介**: 专注于 ArgoCD 和 Flux 的 GitOps 高级工作流，涵盖多集群部署与机密管理 (Secret Management)。
    - **标签**: `GitOps`, `ArgoCD`, `Flux`

### Infrastructure & Pipelines
- [**iac-terraform**](https://github.com/ahmedasmar/devops-claude-skills/tree/main/skills/iac-terraform)
    - **简介**: 协助创建、验证和管理 Terraform/Terragrunt 配置，提供代码审查与状态文件故障排除。
    - **标签**: `Terraform`, `IaC`
- [**ci-cd**](https://github.com/ahmedasmar/devops-claude-skills/tree/main/skills/ci-cd)
    - **简介**: CI/CD 流水线设计与安全加固，支持 GitHub Actions 和 GitLab CI 的调试与优化。
    - **标签**: `CI/CD`, `GitHub Actions`

### Observability (可观测性)
- [**monitoring-observability**](https://github.com/ahmedasmar/devops-claude-skills/tree/main/skills/monitoring-observability)
    - **简介**: 指导 Prometheus/Grafana 搭建，涉及 SRE 核心概念（SLO/错误预算）及 OpenTelemetry 分布式追踪。
    - **标签**: `Monitoring`, `Grafana`, `SRE`
